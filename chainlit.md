# MultiModal RAG

Welcome! This system lets you chat with your documents using AI.

## Supported formats
- **PDF** — text + embedded images
- **Images** — PNG, JPG, TIFF, WebP (described by GPT-4o Vision)
- **Word** — DOCX with text, tables, and images
- **PowerPoint** — PPTX slide content and images
- **Excel / CSV** — spreadsheet data as searchable tables
- **Web URLs** — paste any URL to scrape and ingest

## How to use
1. Upload a file using the attachment button, or paste a URL
2. Wait for ingestion to complete
3. Ask questions in plain English

## Tips
- You can upload multiple files across different formats
- Images inside PDFs and DOCX files are automatically described
- Sources and relevance scores are shown with every answer
